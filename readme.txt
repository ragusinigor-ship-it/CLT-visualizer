Визуализация центральной предельной теоремы (ЦПТ)

Этот проект демонстрирует центральную предельную теорему — одно из ключевых понятий математической статистики.

  Центральная предельная теорема утверждает:  
  Распределение средних значений случайных выборок стремится к нормальному распределению,  
  даже если исходные данные взяты из совершенно другого распределения (например, гамма-распределения).

В проекте показано, как форма распределения средних меняется в зависимости от размера выборки.

Что показывает программа?

1. Исходное распределение — в примере используется гамма-распределение.
2. Распределение выборочных средних при размере выборки n = 5
3. Распределение выборочных средних при размере выборки n = 50.

Чем больше размер выборки — тем ближе распределение средних к идеальному «колоколу».
